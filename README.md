修复了没有默认滴答音的问题。增大了滑动组件（用于调整音量）的可交互范围。更新了一些链接和文字。

公开介绍的第一个版本，更新如下：

- 内置更多免费音效
- 支持切换滴答音效
- 支持调整声音音量
- 提高功能的完整性
